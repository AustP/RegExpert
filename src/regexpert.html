  • This application was made for those who are already familiar with JavaScript Regular Expressions and wish to test theirs.

    Format RegExps like: /[a-z]/g
  • Anchors
  • ^Start of string, or start of line in multi-line pattern
  • $End of string, or end of line in multi-line pattern
  • \bWord boundary
  • \BNot word boundary
  • Character Classes
  • \cControl character
  • \sWhite space
  • \SNot white space
  • \dDigit
  • \DNot digit
  • \wWord character
  • \WNot word character
  • Assertions
  • ?=Lookahead assertion
  • ?!Negative lookahead
  • Quantifiers
  • *0 or more
  • +1 or more
  • ?0 or 1
  • {3}Exactly 3
  • {3,}3 or more
  • {3,5}3, 4, or 5
  • Add a ? to a quantifier to make it ungreedy.
  • Common Metacharacters
  • ^ [ ] . $ { } * ( ) \ + | ? < >
  • The escape character is usually the backslash \
  • Special Characters
  • \nNew line
  • \rCarriage return
  • \tTab
  • \vVertical tab
  • \fForm feed
  • \xxxOctal character xxx
  • \xhhHex character hh
  • \uhhhhUnicode character with hex code hhhh
  • Groups and Ranges
  • .Any character except new line (\n)
  • (a|b)a or b
  • (...)Group
  • (?:...)Passive (non-capturing) group
  • [abc]Range (a or b or c)
  • [^abc]Not a or b or c
  • [a-q]Letter from a to q
  • [A-Q]Upper case letter from A to Q
  • [0-7]Digit from 0 to 7
  • \xxth group/subpattern
  • Ranges are inclusive.
  • Pattern Modifiers
  • gGlobal match
  • iCase-insensitive
  • mMultiple lines
